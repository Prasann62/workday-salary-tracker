# Workday Salary Tracker

A modern, responsive web application built with React and Vite to track daily work hours, calculate salary, manage entries, and visualize earnings over time.

## Features

- **Daily Entry Tracking**: Easily log your daily work hours and calculate earnings based on a configurable hourly wage.
- **Calendar View**: Manage and visualize your work entries on a full-featured calendar using `@fullcalendar`.
- **Dashboard & Insights**: View your earnings, compare statistics, and analyze trends mapped out with `recharts`.
- **Responsive Design**: A beautiful, adaptable UI built entirely with Tailwind CSS, ready for both mobile and desktop.
- **Modern Stack**: Bootstrapped with Vite for fast builds and React 18 for seamless UI interactions.

## Technologies Used

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [FullCalendar](https://fullcalendar.io/)
- [Recharts](https://recharts.org/)
- [Lucide React](https://lucide.dev/)
- [Framer Motion](https://www.framer.com/motion/)

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Prasann62/workday-salary-tracker.git
   cd workday-salary-tracker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
